Profit and Loss Report With Group Features

This module allows creating multi-level profit and loss report with account group features.

Dependencies
  • custom_report.
  • account.
  • account_group.
Quick View
  • App create a menu named 'Profit and Loss with Group' Under 'Accounting/Reports'.
  • Before Creating the report, Account Group needs to be configured.
  • To configure account group, Go to > "Accounting/Configuration/Accounting/Accounts Groups" and Create Groups.
  • At leas 3 Groups needs to be there with pre-fix 4, 5 and 6 named Income, Cost of Revenue and Expense. Rest of the groups will the child of these three Groups. We create multi-level child group.
  • On wizard, date range and Group Level wise filter available. Also we can see debit and credit details
  • Report have both html and pdf view.
